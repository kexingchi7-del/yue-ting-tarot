# 月庭塔罗

一个可本地运行的塔罗牌与八字排盘小程序，支持完整 78 张塔罗牌库、正逆位、三种牌阵、历史记录、复制解读，以及专业历法口径的四柱、大运和流年计算。

## 八字排盘口径

- 支持公历 1900–2100 年。
- 年柱以立春为界，月柱以十二“节”为界。
- 默认采用子初 23:00 换日。
- 真太阳时包含标准经线与出生地经度差，以及均时差修正。
- 大运采用“阳男阴女顺排、阴男阳女逆排”和“三日折一年”口径。
- 1986–1991 年中国出生资料应确认当时是否采用夏令时，并在页面选择对应修正。
- 传统命理存在流派差异，排盘结果用于文化研究与自我探索，不构成科学预测或重大决策建议。

历法计算使用本地锁定的 `lunar-javascript 1.7.7`（MIT），网页不依赖第三方 CDN。

## 使用

启动本地服务：

```bash
npm start
```

然后访问 `http://localhost:4173`。
